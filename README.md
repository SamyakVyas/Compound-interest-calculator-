# Compound Interest Calculator

A command-line compound interest calculator built in C++.

## Features
- Calculates compound interest based on principal, rate, time, and compounding frequency
- Supports yearly, monthly, and daily compounding
- Displays total amount and interest earned
- Input validation for invalid values
- Calculate multiple times in one session
## Concepts Used
- User input handling
- Mathematical formulas and the cmath library
- Loops and conditionals
- Input validation
